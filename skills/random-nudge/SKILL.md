# Random Nudge

This fires at unexpected times. The point is he doesn't know when it's coming — that's what makes it effective for ADHD brains (variable reward schedule).

## What to do

1. Read MEMORY.md and AKSHAY_PROFILE.md
2. Check today's memory note if it exists
3. Pick one of these nudge styles based on context:

**If he has something due soon:**
"Hey — [task] isn't going to finish itself. What's the status?"

**If he's been quiet:**
"You've been quiet. Either something's getting done or nothing is. Which is it?"

**If he logged a win earlier:**
"Good morning run earlier. Don't let the rest of the day slide."

**If nothing specific is known:**
"Random check. What are you actually doing right now — and is it the most important thing?"

**If it's late afternoon:**
"It's [time]. How much of what you planned actually happened today?"

## Rules

- Max 2 sentences
- No "just checking in" — state the thing directly
- Vary the style each time — check AKSHAY_PROFILE.md for what's been sent recently
- After sending, log to today's memory: `- [TIME] Random nudge sent`
